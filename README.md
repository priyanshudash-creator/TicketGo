# 🚆 TicketGo – Train Ticket Booking Website

TicketGo is a simple and user-friendly train ticket booking website built using **HTML**. The project simulates an online railway reservation system where users can browse trains, book tickets, view bookings, cancel reservations, and explore special offers through a clean and organized interface.

---

## 📌 Features

- 🏠 Home Page
- 🔍 Search Available Trains
- 🎫 Book Train Tickets
- 📖 View My Bookings
- ❌ Cancel Ticket
- ✅ Booking Confirmation Page
- 👤 User Registration
- 🔐 User Login
- 🎁 Offers Section
- ℹ️ About Us
- 📞 Contact Us
- 🚆 Individual Train Information Pages

---

## ▶️ How to Run

1. Download or clone this repository.

```bash
git clone https://github.com/your-username/TicketGo.git
```

2. Open the project folder.

3. Double-click **Index.html** or open it in your preferred web browser.

## 🎯 Project Purpose

This project was created to practice front-end web development using HTML by designing a realistic train ticket booking website. It focuses on website structure, navigation, forms, tables, hyperlinks, and user-friendly page layouts.
